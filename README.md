# Simple Calculator
 A simple calculator in [ReactTS + Vite](https://vitejs.dev/) and run with [bun](https://bun.sh/)

## Getting Started
- Install libraries by using ```bun install```
- Start the app by using ```bun run dev```

## Libraries

- postcss - [Link](https://postcss.org/)
